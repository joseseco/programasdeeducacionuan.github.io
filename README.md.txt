# Modelo de Aprendizaje - Página Web

Este es un proyecto sencillo en HTML y CSS que presenta un modelo de aprendizaje dinámico y sinérgico, ideal para mostrar en GitHub Pages.

## Estructura

- **Cabecera** con título y subtítulo.
- **Menú de navegación** responsivo.
- **Contenido principal** dividido en secciones.
- **Botón** para volver al inicio.
- **Pie de página** con derechos reservados.

## Tecnologías

- HTML5
- CSS3 (integrado en el mismo archivo)
